# StrykeX Algos Lifetime – Payment Page

A React landing page for **StrykeX Algos Lifetime**, the automated trading platform by StockWiz. Built with **React**, **Vite**, and **Tailwind CSS**.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Project structure

```
src/
├── components/
│   ├── Header.jsx        # Logo, title, description, feature list
│   ├── DashboardPreview.jsx  # Simulated dashboard UI
│   ├── PaymentForm.jsx   # Payment details form (Razorpay)
│   └── Footer.jsx        # Share, contact, disclaimer, terms
├── App.jsx
├── main.jsx
└── index.css
```

## Stack

- **React 18** – UI
- **Vite** – dev server and build
- **Tailwind CSS** – styling
